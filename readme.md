# TWRP 3.7.0 for LAVA LXX521 (MT6878)

-
This is a custom build of **Team Win Recovery Project (TWRP)** specifically for the **LAVA LXX521** powered by the **MediaTek Dimensity (MT6878)** chipset.

---

## ⚠️ STATUS: ALPHA / BETA
**This is a testing release.** While core functions work, some hardware-specific features may require additional patches (see the Hardware Fix Pack below). Use at your own risk.

### ✅ Working
- [x] ADB / MTP
- [x] Internal Storage Decryption (FBE)
- [x] Touchscreen & Display
- [x] Backup / Restore (Basic partitions)
- [x] Flashing GSI / Flashable ZIPs

### ❌ Issues / WIP
- [ ] Fingerprint (Requires vendor library links)
- [ ] Stock Camera Integration (Fixable via patch)
- [ ] Game Booster Support (Fixable via patch)
